异常的传递

  • 当函数/方法执行出现异常，会将异常传递给函数/方法的调用一方
  • 如果传递到主程序，仍然没有异常处理，程序才会终止

    开发中可以在主函数中增加异常捕获，这样就可以大大减少异常捕获代码，使代码整洁
