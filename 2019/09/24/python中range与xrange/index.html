python中range与xrange

发表于 2019-09-24 分类于 python
本文字数： 523 阅读时长 ≈ 1 分钟
在python2中存在`xrange()`，python3中`range()`的实现方式与`xrange()`相同，所以就不再有`xrange()`

用法

  • range(start, stop, step)与xrange(start, stop, step)用法相同
    • start：起始数字，可不填，默认0
    • stop：结束数字，必须填
    • step：步长，可不填，默认1

区别

  • range()直接生成一个列表，需要立即开辟内存空间
  • xrange()生成一个可以迭代的对象，即用即取
  • 要生成很大的数字序列的时候，用xrange会比range性能优很多，因为不需要一上来就开辟一块很大的内存空间
